# Dungeon Witch

An urban witch dungeon crawler written in PyGame.

## Authors

Girl Void: Olivia Rhodes, Madisyn DelPorto, Briar Rose Schreiber

## Setup

* Make a virtual environment for Python 3.6
* Run `pip install -r requirements.txt`
